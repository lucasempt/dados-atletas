# dados-atletas
Este projeto foi desenvolvido como solução para o desafio proposto no curso DEVstart, onde o objetivo é criar uma aplicação em JavaScript capaz de receber informações de um atleta e calcular:

- Categoria do atleta (conforme faixa etária)
- IMC (Índice de Massa Corporal)
- Média válida das notas (utilizando o método do Projeto de Certificação 1)
- Estrutura do Projeto

O projeto contém:

- Uma classe `Atleta` responsável por armazenar atributos e disponibilizar métodos para todos os cálculos necessários.
- Um exemplo de uso ao final do arquivo `dados-atletas.js`, demonstrando como instanciar um atleta e exibir seus dados.

---

Atributos da Classe Atleta

A classe recebe os seguintes parâmetros:

- `nome` — Nome do atleta
- `idade` — Idade em anos
- `peso` — Peso em kg
- `altura` — Altura em metros
- `notas` — Array contendo as notas atribuídas ao atleta
